What is ROCA?

ROCA is a collection of recommendations for decent Web application frontends. You can read more about it on roca-style.org. This application serves as an example how the ROCA recommendations can be applied using the Play Framework. Each section of this tutorial refers to one of the ROCA principles and describes how it is applied by the example application.
